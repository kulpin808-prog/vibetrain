# 🚀 Настройка Telegram Web App

## 📋 Что такое Web App?

Telegram Web App - это веб-приложение, которое работает прямо внутри Telegram. Пользователи могут взаимодействовать с ним без перехода в браузер.

## 🎯 Преимущества для нашего бота:

- **📱 Мобильный интерфейс** - удобная форма для ввода данных
- **🎨 Красивый дизайн** - современный UI с анимациями
- **⚡ Быстрый ввод** - чекбоксы, радиокнопки, слайдеры
- **🤖 Интеграция** - автоматическая отправка данных боту

## 🛠️ Настройка Web App

### Шаг 1: Разверните сервер

Сначала нужно развернуть Web App на сервере (Railway, Heroku, etc.):

```bash
# Запуск локально для тестирования
npm run webapp

# Web App будет доступен по адресу:
# http://localhost:3000
```

### Шаг 2: Настройте в BotFather

1. **Откройте** [@BotFather](https://t.me/botfather)
2. **Выберите бота** командой `/mybots`
3. **Выберите** "Fitness Trainer AI Bot"
4. **Выберите** "Bot Settings" → "Configure Mini App" (или "Web Apps")
5. **Если Mini App отключен**, нажмите "Enable Mini App"
6. **Отправьте ссылку** на ваш Web App: `https://your-domain.com`

### Шаг 3: Обновите переменные окружения

Добавьте в Railway/Heroku переменную:
```
WEBAPP_URL=https://your-railway-app.com
```

### Шаг 4: Проверьте работу

После настройки:
1. Напишите боту `/start`
2. Нажмите "📝 Создать программу"
3. Должно открыться ваше Web App
4. Заполните форму и отправьте
5. Бот должен получить данные и создать программу

## 🧪 Тестирование Web App

### Локальное тестирование:

1. **Запустите сервер:**
   ```bash
   npm run webapp
   ```

2. **Откройте в браузере:**
   ```
   http://localhost:3000
   ```

3. **Заполните форму** и нажмите "Создать программу"

### Тестирование в Telegram:

1. **Напишите боту:** `/start`
2. **Нажмите кнопку:** "📝 Создать программу"
3. **Заполните форму** в Web App
4. **Получите** готовую программу!

## 🎨 Кастомизация Web App

### Изменение дизайна:

- **Цвета:** редактируйте `public/styles.css`
- **Текст:** изменяйте `public/index.html`
- **Логика:** обновляйте `public/script.js`

### Добавление новых полей:

1. **Добавьте HTML** в `index.html`
2. **Добавьте обработку** в `script.js`
3. **Обновите** функцию `buildWorkoutRequest()`

## 📱 Структура Web App


### Пример расширения:

```javascript
// Добавить сохранение в localStorage
function saveUserPreferences(data) {
    localStorage.setItem('fitness_prefs', JSON.stringify(data));
}

// Загружать сохраненные настройки
function loadUserPreferences() {
    const saved = localStorage.getItem('fitness_prefs');
    return saved ? JSON.parse(saved) : {};
}
```

## 🚀 Развертывание Web App

### Рекомендуемый способ (Railway):

1. **Создайте аккаунт** на [railway.app](https://railway.app)
2. **Подключите GitHub** репозиторий
3. **Railway автоматически** обнаружит Node.js проект
4. **Добавьте переменные:**
   - `NODE_ENV=production`
   - `WEBAPP_URL=https://your-app-name.up.railway.app`

### Альтернативы:

- **Vercel:** `vercel --prod`
- **Heroku:** `git push heroku main`
- **DigitalOcean:** App Platform

## 🐛 Устранение проблем

### Web App не открывается:
- Проверьте URL в BotFather
- Убедитесь, что сервер запущен
- Проверьте логи Railway

### Данные не отправляются:
- Проверьте `console.log` в браузере
- Убедитесь, что `tg.sendData()` вызывается
- Проверьте обработчик в `index.js`

### Стили сломались:
- Проверьте CSS в мобильном браузере
- Добавьте `viewport` meta tag
- Используйте flexbox вместо grid для совместимости
