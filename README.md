# VibeTrain

Минимальное приложение: статика и HTTP.

```bash
npm install   # создаёт package-lock.json
npm start
```

Откройте [http://localhost:3000](http://localhost:3000) (порт — `PORT`). Превью в макете телефона; полный экран в браузере: [?mock=0](http://localhost:3000/?mock=0). В Telegram Web App макет отключается автоматически.

Прототип **админки клуба** (мок-данные): **http://localhost:3000/admin/** (нужен запущенный `npm start`; в консоли должны быть строки `Web:` и `Admin:`).

Если в браузере **`{"error":"Not found"}`** — на этом порту отвечает **другой** сервер (старый код без `/admin/` или другой процесс). Остановите лишние `node`, проверьте `lsof -i :3000`, перезапустите `npm start` из этого репозитория; в логе должны быть строки `Web:` и `Admin:`.

После `npm install` автопроверка маршрутов (включая `/admin/`): `npm run verify`.
